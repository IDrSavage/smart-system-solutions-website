# Smart System Solutions — الموقع الإلكتروني

موقع تعريفي عصري (صفحة واحدة، عربي RTL) لمؤسسة **Smart System Solutions**
لحلول الاتصالات وتقنية المعلومات والحلول البرمجية.

🔗 **معاينة مباشرة:** _(يُضاف رابط النشر هنا)_

## المميزات
- 🎨 تصميم عصري بهوية الشركة (تدرّج بنفسجي / طابع تقني) مع خلفية شبكة متحركة.
- 🌗 **وضعان**: ليلي ونهاري مع زر تبديل (يتذكّر اختيار الزائر).
- 🧩 **17 خدمة** مدمجة (اتصالات وتقنية + حلول برمجية) مع فلترة حسب النوع.
- 🖼️ معرض أعمال مع عارض صور (Lightbox).
- 📝 نموذج تواصل يرسل الطلبات إلى **Google Sheets** + إشعار بالبريد.
- 💬 زر واتساب عائم للتواصل المباشر.
- 📱 متجاوب بالكامل (جوال / تابلت / حاسب).
- ⚡ بدون أطر عمل — HTML + CSS + JavaScript نقي، سريع وخفيف.

## البنية
```
index.html              # الصفحة
assets/css/styles.css   # التنسيقات (مع متغيرات الثيم)
assets/js/main.js       # التفاعلات (الخدمات، المعرض، النموذج، الثيم)
assets/img/             # الشعارات وصور المعرض
GOOGLE-SHEET-SETUP.md   # خطوات ربط النموذج بـ Google Sheets
```

## التشغيل محلياً
أي خادم ثابت يكفي:
```bash
python -m http.server 8000
# ثم افتح http://localhost:8000
```

## ربط نموذج التواصل
راجع [GOOGLE-SHEET-SETUP.md](GOOGLE-SHEET-SETUP.md) لربط النموذج بـ Google Apps Script.
